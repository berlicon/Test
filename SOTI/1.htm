Last Man Standing
 
Consider this game "last man Standing":
N people are standing in form of a circle. Every mth person gets eliminated in consecutive turns.
Write a function to return the value of the last man standing, given the count N and the elimination factor m. 
 
Input Specification:
N: integer value N where  1<N<109
m: integer value m i.e elimination factor where 1<m<10 9 and m<N
 
Output Specification:
Return the position of the last man standing.
 
Example 1:
N: 5
m: 2
 
Output: 3
 
Explanation:
First, 2 gets eliminated, next 4 then 1, then 5 and the last man remaining is 3.

N: 6
m: 3
 
Output: 1
